## How to use this file

Every pattern is a plug-and-play template with variables in `{curly braces}`. Paste these into code as the subject line, preheader, and copy generators for the newsletter cron.

Variables you'll see across patterns:

- `{childName}` — first name only, only if `child_name_consent = true`
- `{subject}` — one-word descriptor of the photo (boots, bath, beach, bike)
- `{occasion}` — birthday, first day of school, Mother's Day, etc.
- `{pageCount}` — 30, 50, or 100
- `{weekNumber}` — calendar week (for the "this week's" pattern)
- `{number}` — a small specific integer (pages shipped, new families, etc.)
- `{dayOfWeek}` — Monday, last Tuesday, etc.
- `{momentTag}` — customer-selected context tag

## Archetype A — Sunday Show-Off

### Subject Line Patterns (10+)

All target ≤ 42 characters. Narrative tone. One family, one photo, one moment.

```
1.  This week's page: {childName} and the {subject}
2.  What {childName}'s mom printed this weekend
3.  The {subject} photo that became a page
4.  {childName}, {approxAge}, in the coloring book
5.  A {momentTag} photo, now a coloring page
6.  From the camera roll: {childName}'s {subject}
7.  Meet this week's featured family
8.  The {occasion} photo we turned into a page
9.  One photo. One page. One family.
10. {childName}'s page is in the {pageCount}-page book
11. Last week's favorite upload
12. What showed up in {childName}'s coloring book
```

### "What Makes This Work" Micro-Caption Patterns (5)

One sentence. The smart-friend voiceover.

```
1. "The high-contrast light made {subject} and {possessive} profile
   translate almost one-to-one."

2. "A clear face and a single prop — the {subject} — is often all
   a great page needs."

3. "Overcast days actually turn into better pages than bright sun."

4. "A photo with one subject in focus beats a busier scene every time."

5. "Side profiles and action shots both translate well — the page
   keeps the motion."
```

### P.S. Patterns (5)

   another family."

2. "P.S. {number} grandma copies went out Friday alone. See
   you Thursday for the gallery."

4. "P.S. {number} families got their books last week.
   Next Sunday: one of yours, maybe."

5. "P.S. {number} pages. Zero screens. That was this week."

### Prompt of the Week Copy Patterns (12 — one per week, one quarter)

Each is a single card. The "take a photo of..." verb stays consistent so the format is learnable.

```
1.  "Take a photo of your kid doing their favorite messy thing —
     mud, paint, spaghetti sauce, the works."

2.  "Take a photo of the face they make when they're focused on
     something. Legos, drawing, a caterpillar — doesn't matter."

3.  "Take a photo of your kid in their favorite outdoor spot.
     Not posed. Just there."

4.  "Take a photo of them and their grandma. Bonus: same chair,
     years apart."

5.  "Take a photo of the exact moment before a tantrum.
     Come on. You have one."

6.  "Take a photo of your kids being uncharacteristically nice
     to each other. (Document it. You'll forget.)"

7.  "Take a photo of your kid's 'I built this' moment.
     Blocks, sand, pillow forts — the proud one."

8.  "Take a photo with the pet and the kid in the same frame.
     These translate beautifully."

9.  "Take a photo from bath time (PG, obviously — the goggles,
     the rubber duck, the wet hair)."

10. "Take a photo of the costume they've been wearing at home
     every day for six weeks straight."

11. "Take a photo from behind. Their favorite spot.
     Back-of-the-head shots make gorgeous pages."

12. "Take a photo of a car-seat nap. Cheek smushed on the strap.
     That one."
```

## Off-Brand Anti-Patterns — Never Write These

### Banned verbs (appear nowhere in any newsletter)

```
✗ unlock
✗ supercharge
✗ level up
✗ leverage
✗ optimize
✗ activate
✗ crush
✗ dominate
✗ transform (overused; prefer "turn into")
✗ empower
```

### Banned phrases

```
✗ "In today's edition..."
✗ "This week we'll explore..."
✗ "Without further ado..."
✗ "Here's what's coming up..."
✗ "We're so excited to share..."
✗ "Our amazing AI..."
✗ "Our workflow / pipeline / process..."
✗ "Our customers / users / audience..."  → use "moms" or "families"
✗ "Join thousands of parents..."
✗ "Don't miss out!"
```

### Banned structural moves

```
✗ Multiple exclamation points in one email
✗ ALL CAPS subject lines
✗ Emoji in subject lines (a restrained emoji in the body is okay
  rarely — e.g., a single 📬 for the forward line — but NOT default)
✗ A founder photo or founder signature ("— Alec")
  — the customer does not want to hear from the CEO in the newsletter
✗ Discount codes in the newsletter body (those go in dedicated
  promo sends, not in the Sunday show or the Thursday gallery)
✗ Countdown timers
✗ Stock photography anywhere — ever — the whole product is
  real family photos
```

### Banned tone

```
✗ "Hey mama!"   — too gushy, too "mommy blog"
✗ "Friend!"     — LinkedIn fake-warmth
✗ "Hi there,"   — generic
✗ Over-poetic openers that delay the value
✗ Humble-brag metrics dressed as gratitude
✗ Anything that reads like a pitch deck slide
```

## Pattern Freshness Guardrails

A pattern bank only works if it doesn't rerun the same sentence two weeks in a row. The cron should track:

- Which subject line template was used for the last 4 Sundays and last 4 Thursdays. Don't repeat within 4 weeks per archetype.
- Which P.S. template was used for the last 3 Sundays. Don't repeat within 3 weeks.
- Which prompt was used for the last 12 Thursdays. Rotate through all 12 before reusing any.

Log each pick to the send record so a future audit can verify rotation.
