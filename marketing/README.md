# Marketing Ops

This directory holds the OpenClaw marketing automation specs, experiment plans, report outputs, and queue artifacts.

## Files

- `adapter-spec.md`: contracts and orchestration rules for internal tools, Arcads, ElevenLabs, Gamma, and metrics ingestion
- `testing-matrix-30-days.md`: first 30-day content and ad testing plan

## Folders

- `briefs/`: creator briefs, occasion briefs, campaign packs
- `experiments/`: experiment registries, daily batches, and result snapshots
- `queues/`: publish queues and paid nomination outputs
- `reports/`: weekly and monthly synthesis outputs
